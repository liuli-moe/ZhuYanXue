# 第八部 海棠

封面人物：海棠

本部简介：

青竹和山药，是湘西这块穷乡僻壤很能倒腾点钱的两样生计。盛产青竹的地方为数并不多，沅镇算是最出名的一个，似一颗明珠镶在丛山之中，玉带般的沅水绕城而过，城外整山整岭都是竹，风一吹，就哗啦啦响成一片，翠叶起伏连绵不绝，像大海的波浪，一排排，一浪浪，所以人们习惯叫这里是翠竹海。

高洁的官家少妇，沦落成仇人玩物，做过妓女，当过尼姑，到最后，还是逃脱不了永远沦为性奴的命运。
